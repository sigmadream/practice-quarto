Welcome

콰르토로 멋진 문서를 만들어 온라인에서 공유할 수 있고, 이 문서는 GitHub로 자동 업데이트됩니다. 이 튜토리얼은 Quarto 웹사이트의 간단한 예제입니다.

코드 작성 여부와 관계없이 자신이나 동료가 분석에 사용하는 것과 동일한 재현 가능한 워크플로를 사용하여 기술과 과학에 기여할 수 있습니다. Quarto 웹사이트를 만드는 것은 R, Python 또는 HTML, CSS 등을 몰라도 할 수 있습니다만, Quarto는 이러한 도구와 통합되므로 오픈 커뮤니티에서 예제를 보고 다른 사람의 예제를 재사용 할 수 있습니다.

저는 글쓰기를 한 마디로 “정직한 글쓰기, 빠른 배포, 감사한 관용”이라 주장합니다. Quarto는 이러한 제 주장에 어울리는 도구입니다.
